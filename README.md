# Personal Finance Manager with AI

An AI-driven personal finance manager that uses time-series forecasting, OCR, and NLP-based classification to automate spending tracking.


## Problem Description
Manual personal expense management takes a lot of time and is prone to mistakes. 
People frequently neglect to keep track of their expenditures across transactions and receipts, which results in inadequate financial planning and budgeting.

## Objective
to create a comprehensive system that automatically extracts, classifies, evaluates, and forecasts personal spending.

## Planned Tech Stack: Python; Machine Learning (NLP, Time Series); OCR; FastAPI; Streamlit
